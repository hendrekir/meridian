SDK · 2-minute setup

One line. That's it.

Wrap your existing Anthropic or OpenAI client. Every call is automatically timed, costed, and attributed — in a background thread that adds zero latency to your app.

1
Install
One package. Zero dependencies. Python 3.9+.
2
Wrap your client
Change one line. Streaming, async, retries — all stay identical.
3
See your margins
Live per-user, per-feature gross margin. Real-time. No deploy needed.
# 1. Install pip install meridian-sdk # 2. Wrap your client — one line change from meridian_sdk import MeridianProxy import anthropic # Before: client = anthropic.Anthropic() client = MeridianProxy( anthropic.Anthropic(), api_key="mrd_your_key_here", feature="ai-chat", # maps to Feature Margins view user_id=req.user.id # maps to per-user profitability ) # 3. Use exactly as before — nothing else changes msg = client.messages.create( model="claude-sonnet-4-6", max_tokens=1024, messages=[{"role": "user", "content": prompt}] ) # Cost recorded in background. Zero latency added.
Also works with openai.OpenAI() — same wrapper, same one-line change.
